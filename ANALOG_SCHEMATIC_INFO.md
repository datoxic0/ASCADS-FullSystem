# Analog Schematic Editor Info

The Analog Schematic Editor is a sprawling canvas for high-fidelity component routing, offering an advanced engine for deep physical modeling. 

### Key Features
- **Dynamic Physics Engine**: Real-time evaluation of nodes, voltages, and currents.
- **Parametric Components**: Double-click any component (resistors, capacitors, voltage sources) to adjust its physical parameters.
- **Live Wiring**: Draw wires dynamically between connection ports. Green indicates a valid ground path or logic high, while blue indicates neutral or logic low.
- **Simulation Control**: Run, pause, and step through transient simulations directly from the toolbar or by using the global `Spacebar` shortcut.

### Controls
- **Drag**: Left click and drag to move components.
- **Wire**: Click on a component port (small circle) and drag to another port to create a wire.
- **Delete**: Select a component or wire and press `Delete` or `Backspace`.
- **Properties**: Double click a component to open its properties menu.

# ASCADS (Advanced Schematic Design Full System)

ASCADS is an advanced, multi-modal web-based engineering environment integrating 3D CAD modeling, Digital/Analog Schematic Simulation, Robot Kinematics, and Programmable Logic Controller (PLC) systems into a single seamless interface.

## Features

- **EngiGraph 3D Stability**: Stable WebGL-based CSG engine capable of importing standard `.stl` and `.obj` files alongside dynamically generated script geometry.
- **Hardware Bus Bridging**: True cross-module synchronization using a shared hardware bus (PLC ladder logic drives robot sensors and vice-versa, running simultaneously in the background).
- **Advanced Analog Matrix**: Realistic node-voltage propagation solver equipped with a live waveform logic analyzer and oscilloscope tools that update continuously at 60fps.
- **AI Copilot Modes**: Toggle between high-fidelity Power AI engines and optimized Economy AI engines for code generation and 3D environment awareness.

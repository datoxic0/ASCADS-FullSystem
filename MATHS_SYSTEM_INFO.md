# Maths & Compute Tools Info

ASCADS comes with an advanced mathematics and computation suite that allows for both standard base conversion operations and high-level abstract mathematical manipulation.

### Included Tools

#### 1. Binary & Logic Calculators
- **Base Converter:** Seamlessly translate between Decimal, Binary, Octal, and Hexadecimal representations, including fractional precision and signed two's complement.
- **Binary Arithmetic:** Perform binary addition, subtraction, multiplication, and division as well as bitwise logic (AND, OR, XOR, etc.).
- **Boolean Logic Lab:** An interactive Truth Table that dynamically synthesizes boolean expressions, minimizing them via K-map logic and rendering interactive gate schematics.

#### 2. Advanced Mathematics
- **Complex Numbers:** Input real and imaginary values to compute magnitude, phase angle, and standard arithmetic operations on complex planes.
- **Matrix Operations:** Compute the product, addition, inverse, or determinant of user-defined matrices.
- **Fourier Series Analysis:** Visualize the reconstruction of Square, Sawtooth, and Triangle waves from their harmonic sine components.
- **3D Vector Geometry:** Calculate dot products, cross products, magnitudes, and angles for spatial vectors.

#### 3. Electrotechnology
- Calculate Ohm's Law and Power equations.
- Determine the Resonant Frequency and Impedance of RLC circuits.
- Perform Three-Phase Power calculations.
- Analyze Transformer turns ratios and step-up/step-down parameters.

### Usage
- Select a tool from the side panel to begin computing.
- Most calculators update in real time; for more advanced tools, select the specific operation (e.g. `Det(A)` for matrix determinant).
- The Information icon on each tool provides localized context about limits and logic bounds.
